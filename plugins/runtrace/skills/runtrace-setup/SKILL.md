# RunTrace setup

Prepare one reliable tracking path before experiments begin.

- Read [references/preflight.md](references/preflight.md) for repository integration or readiness checks.
- Read [references/connection.md](references/connection.md) for installation, authentication, or connection failures.
- Do not start an experiment during setup unless the user also requested execution.
- Never expose credentials or create a second run to test an existing tracking path.
